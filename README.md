Roof Yard Restaurant Website

A modern dark-themed restaurant website inspired by popular food ordering platforms.

Features

* Dark modern UI
* Responsive mobile-friendly design
* Category-based menu navigation
* Starters, Desi, Pasta & Sandwiches, Burgers, Chinese, BBQ, and Rolls sections
* Add-to-cart functionality
* Automatic cart total calculation
* Restaurant-style menu cards
* Easy menu management
