# Roof--Yard--Restaurant
A modern food ordering website built for Roof Yard (Karachi).
## 🔥 Features
- Full menu system (Starters, BBQ, Karahi, Burgers, Drinks)
- Image-based UI
- Add to cart system
- Half / Full portion selector
- WhatsApp ordering system
- Mobile friendly design
- Dark rooftop flame theme
## 📍 Restaurant Info
- Name: Roof Yard
- Location: Plot no. SF 9, Block E North Nazimabad Town, Karachi
- Phone: 03196639273
- WhatsApp: +92 319 6639273
## 🚀 How it works
Users select items → add to cart → place order → WhatsApp opens with order message.
